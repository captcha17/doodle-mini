# Doodle Mini (Meeting Scheduling Service)

Meeting scheduling platform using Spring Boot and Java technologies

## Overview

The service enable users to manage their time slots, schedule meetings, and view their custom calendar availability.

## Features

- Users are able to define available slots, which can later be converted into meetings
- Each user have a personal calendar where their time is managed
- A slot can be booked as a meeting with a specific title and participants
- The system supports querying free or busy slots, providing an aggregated view for a selected time frame
